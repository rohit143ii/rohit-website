Lesson 1: Introduction to JavaScript

JavaScript ek programming language hai jo website ko interactive banata hai. Ye HTML aur CSS ke sath milkar web pages ko powerful banata hai. Example ‚Äì jab tum button pe click karte ho, alert box ya animation aata hai ‚Äì wo sab JavaScript se hota hai. 

console.log("Hello World!");

Lesson 2: Variables and Data Types

Variable ek container hota hai jisme data store hota hai. JavaScript me teen main keyword hote hain: var, let, aur const. 

let name = "Rohit";
const age = 18;
var city = "Madhya Pradesh";

Data types: String, Number, Boolean, Array, Object.

Lesson 3: Operators in JavaScript

Operators ka use calculation ya comparison ke liye hota hai.


let a = 10;
let b = 5;
console.log(a + b); // Addition
console.log(a - b); // Subtraction
console.log(a * b); // Multiplication
console.log(a / b); // Division

Lesson 4: Conditional Statements (if-else)

Ye code me decision lene ke kaam aata hai.


let marks = 85;

if (marks >= 80) {
    console.log("Excellent");
} else if (marks >= 50) {
    console.log("Good");
} else {
    console.log("Need Improvement");
}

Lesson 5: Loops in JavaScript

Loop ka kaam same code ko repeat karna hota hai jab tak condition true rahe.


for (let i = 1; i <= 5; i++) {
    console.log("Hello " + i);
}

Lesson 6: Functions

Function ek reusable block hota hai jo specific task perform karta hai.


function greet(name) {
    console.log("Hello, " + name + "!");
}

greet("Rohit");

Lesson 7: Arrays and Objects

Array ek list jesa data structure hai aur Object ek real-world item jesa.


let fruits = ["Apple", "Banana", "Mango"];
let person = { name: "Rohit", age: 18, city: "Shajapur" };

Lesson 8: DOM Manipulation

DOM (Document Object Model) se hum HTML elements ko control karte hain.


document.getElementById("title").innerText = "Welcome Rohit!";

Lesson 9: Events in JavaScript

Event ek action hota hai jaise button click, mouse hover, ya key press.


function showMessage() {
    alert("Button clicked!");
}

HTML me: <button onclick="showMessage()">Click Me</button>

Lesson 10: ES6 Features

ES6 me naye features aaye jaise Arrow Functions, Template Literals, Classes, etc.


const add = (a, b) => a + b;
console.log(`Sum is ${add(5, 3)}`);
